# WordPress Plugins Collection

This repository contains multiple custom WordPress plugins.  
Each plugin lives in its own folder and can be installed independently.

---

## 📂 Included Plugins

### 1. [KnowledgeBase Author Reviewer (KBAR)](./knowledgebase-author-reviewer)
Adds author and reviewer boxes with multiple layout options to posts.  
Ideal for documentation, blogs, or knowledgebase content.

### 2. [Mental Health Tools](./mental-health-tools)
Provides mental health resources and interactive tools for WordPress sites.  
Useful for mindfulness, stress management, and wellness support.

---

## Installation

1. Download the plugin folder you need.  
2. Upload it to your WordPress site's `wp-content/plugins/` directory.  
3. Activate the plugin via the WordPress dashboard.

---

## Notes

- Each plugin includes its own detailed `readme.txt`.  
- Plugins are independent and can be used separately.
